# ClassBuzz - Classroom Management Platform

ClassBuzz is a beautiful, interactive, and modern classroom management platform designed to connect teachers and students in real time. It facilitates streamlined communication, interactive polls, collaborative Q&A (doubts), deadline tracking, resource sharing, feedback loops, and gamified engagement with badges and leaderboards.

## 🌐 Live Demo

**Frontend Deployment:** https://class-buzz-classroom-management-pla.vercel.app/

Explore the live application to experience announcements, polls, doubt management, deadline tracking, resource sharing, gamification, and classroom collaboration features.

## 📈 Project Highlights

* Built using a modern full-stack architecture with React, TypeScript, Node.js, Express, and MongoDB.
* Role-based authentication for teachers and students using JWT.
* Real-time classroom engagement through polls, announcements, and discussion boards.
* Interactive dashboard experience with charts, leaderboards, and activity tracking.
* Responsive and modern user interface powered by TailwindCSS and Shadcn UI.
